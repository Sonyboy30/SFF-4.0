# Soni Family Foundation Website

Static website built for GitHub Pages.

## Pages
- Home: `index.html`
- About: `about.html`
- Our Work: `projects.html`
- Partnerships: `community.html`
- Impact: `impact.html`
- Get Involved: `get-involved.html`
- Updates: `updates.html`
- Support Us: `donate.html`

## Publish
Upload the contents of this folder to the repository branch used by GitHub Pages. Keep `CNAME` in the root so the custom domain stays connected.

## Edit
- Page copy: HTML files in the root
- Colors and layout: `css/style.css`
- Mobile menu and scroll effects: `js/main.js`
- Photos and logo: `assets/`
